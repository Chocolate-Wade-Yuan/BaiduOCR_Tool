# 智能桌面 OCR 图文识别助手 🔍

## 📝 项目简介
本项目是一个基于 Python 和百度云 AI 开放平台开发的轻量化桌面级 OCR 工具。用户可以通过直观的图形用户界面（GUI），一键选择本地图片，系统将自动调用云端接口，快速提取并展示图片中的文字信息。

## 🚀 核心功能
* **图形化交互**：基于 `tkinter` 构建的极简桌面端界面，操作零门槛。
* **本地文件解析**：支持 `.jpg`, `.jpeg`, `.png`, `.bmp` 等主流图片格式的读取与二进制流转换。
* **高精度图文识别**：深度接入百度智能云“通用文字识别”API，实现毫秒级响应与高精度文字提取。
* **异常处理机制**：内置完善的 API 调用失败或网络错误提示，保障程序健壮性。

## 🛠️ 技术栈
* **开发语言**：Python 3.x
* **GUI 框架**：`tkinter`
* **核心依赖库**：`baidu-aip` (百度 AI 官方 SDK)、`chardet`

## 💡 使用说明
1. 确保本地已安装 Python 环境。
2. 在终端运行以下命令安装必要依赖：
   `pip install baidu-aip chardet`
3. 注册百度 AI 开放平台，获取专属的 `APP_ID`, `API_KEY`, `SECRET_KEY`，并将其填入 `main.py` 顶部的配置区。
4. 运行 `main.py`，在弹出的窗口中选择本地图片即可开始识别。
